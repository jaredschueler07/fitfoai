üõ†Ô∏è Technical Implementation Notes

Navigation Architecture:
Jetpack Navigation Compose with nested navigation graphs for onboarding vs main app flows
State Management:
ViewModels with StateFlow/LiveData, Room database for offline persistence
UI Framework:
100% Jetpack Compose with Material 3 design system and custom athletic theme
Permissions:
Location permissions handled through dedicated permission screen flow
API Integration:
Google Fit API, Spotify Web API, Vertex AI for coaching, ElevenLabs for TTS
Real-time Features:
GPS tracking with FusedLocationProvider, background services for run tracking
